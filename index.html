Saya Adalah...

Saya adalah seorang pemimpi yang sedang menempuh pendidikan disalaha satu sekolah ternama dibogor, yaitu SMK-SMAK Bogor. Sebelumya perkenalkan nama saya Faqih Munawar Azhar, saya biasa dipanggil Faqih atau teman teman saya biasa panggil saya dengan sebutan IMUT. Entah dari mana asal mula panggilan tersebut. Saya lahir di Banten tanggal 05 Agustus 2002, tapi saya besar ditanah sunda, yaitu di Sukabumi.
